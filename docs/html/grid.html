Grid columns

The grid numeration start from 1 to 12, example fy-gcs5

fy-gcxs[1-12] The grid is always active
fy-gcs[1-12] Active the grid only when the screen is > 560px
fy-gcm[1-12] Active the grid only when the screen is > 768px
fy-gcl[1-12] Active the grid only when the screen is > 980px

Grid gutter responsive

Sometimes we want gutter only when the grid is active, add the @l | @m | @s | @xs modifier to make it work

-g@xs Gutter active for all screen sizes
-g@s Gutter active on screen > 560
-g@m Gutter active on screen > 768
-g@l Gutter active on screen > 980
